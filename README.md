# Valentine's Day Wish

A single-page Valentine's Day website built with **Next.js** and **Bootstrap**.

## Flow

1. **Entry page** – Enter "Your name" and "Your Valentine's name".  
   Access is allowed only when the pair is **Jeyantharan** and **Pavithra** (order doesn't matter, case-insensitive).

2. **Home page** – Shows an envelope with an **Open** button.

3. **Letter** – Clicking **Open** plays the envelope animation and reveals a Valentine's Day wish letter.

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build for production

```bash
npm run build
npm start
```

## Tech

- Next.js 14 (App Router)
- React 18
- Bootstrap 5 + React-Bootstrap
- TypeScript
